JupyterLab was created to be the all-in-one workspace for data science, scientific computing, and analytical research. It provides the power, speed and flexibility to get more work done.

  • Made for everyone
  • Extensible
  • Customizable

Dockpanel

In JupyterLab, everything you do is held in one place: the dockpanel.
Drag and drop documents to customize your workspace to make it the best for what you do.
Need to pick up where you last started? No problem, JupyterLab and the dockpanel remember your previous session to get you up and running from launch.

Extensions

Extensions are the easiest way to tailor the JupyterLab experience. Find new extensions through the open source community or even build your own. JupyterLab includes a variety of extensions so that you can hit the ground running.

File Browser

Organizing and accessing your files has never been easier. Quickly find files, see running notebooks, and distinguish between different file types at a glance.

Git

Version control done right. The Git plugin for JupyterLab provides an easy to use interface to help you interact with your latest changes. Navigate to repos through the built-in file browser and the Git plugin will do the rest!

Command Pallete

Take control of your development environment. The command pallete was built to help you find what you need. Open applications, access shortcuts, and much more.

Running Sessions

Convienently access all current running sessions in the dockpanel. Quick access to kernel shutdown makes it easy to take control of your working environment.
