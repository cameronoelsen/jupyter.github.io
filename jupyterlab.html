JupyterLab is the all-in-one workspace for data science, scientific computing, and analytical research. It provides the power, speed and flexibility to get work done.

  • Made for everyone
  • Extensible
  • Customizable

Dockpanel

In JupyterLab, everything you do is held in the dockpanel.
Drag and drop to customize your workspace.
Need to pick up where you last started? The dockpanel remembers your previous session to get you up and running.

Extensions

Tailor your JupyterLab experience with extensions. Find community-contributed extensions with the topic juptyerlab-extensions on GitHub. We also have cookie-cutter templates in Javascript and Typescript to build your own extensions.

File Browser

Organizing and accessing your files has never been easier. Quickly find files, see running notebooks, and distinguish between different file types at a glance.

Git

Version control done right. The Git plugin for JupyterLab provides an easy to use interface to help you interact with your latest changes. Navigate to repos through the built-in file browser and the Git plugin will do the rest!

Command Pallete

Take control of your development environment. The command pallete was built to help you find what you need. Open applications, access shortcuts, and much more.

Running Sessions

Conveniently access all current running sessions in the dockpanel. Quick access to kernel shutdown makes it easy to take control of your working environment.

Cell Tools

Jupyter notebook cells are now more powerful in Jupyterlab. Drag and drop cells, both within a document and between notebooks. Cell inputs and outputs can be collapsed and expanded. The previous workflow of cutting, copying, and pasting cells also works in Jupyterlab.

Tab Manager

View active tabs in the tab manager. In single document view, you can also switch between documents using the tab manager.

Launcher

Launch a new Jupyter notebook, text editor, code console, or terminal in Jupyterlab. The launcher gives you the ability to choose between your available kernels.

Code Console

Jupyterlab contains a Qt-like console compatible with the kernels available in the Jupyter ecosystem.

Built-in Terminal

The terminal allows you to execute commands in the terminal without leaving Jupyterlab.

Built-in Text Editor

Jupyterlab provides a text editor connected that can execute code through the code console.

Themeable

We provide both a light and dark theme for Jupyterlab.

CSV Viewer

CSVs can be viewed in Jupyterlab. Large files load quickly and scroll effortlessly.

Image Viewer

Jupyterlab can preview images from within your file manager.
